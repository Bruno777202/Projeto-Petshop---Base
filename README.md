# petshop amigo fiel

---

## funcionalidade

-**homepage com hero e chamada para eles
-**seção de serviços(banho, tosa, veterinario)

## 🚀 como executar
1.clone este repositario
2.abra a pasta ' projeto-petshop' no seu editor ou nagador
3. pasta abrir o arquivo `index.html` ou seu navegador
